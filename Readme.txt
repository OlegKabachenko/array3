Цей застосунок написаний мовою програмування Python.
Щоб запустити програму в ОС Debian, необхідно відкрити консоль, перейти до каталогу проекту(наприклад cd ./projects/array3),
після чого застосувати команду sh array3.sh.
Після цього, в консолі користувач повинен буде увести розмір списку чисел, та увести значення для кожного елементу списку.
Коли користува уведе всі необхідні данні, застосунок покаже:
 1) Мінімальне/максимальне значення списку.
 2) Виведе список, де кожен елемент це значення оригінального списку, яке поділене на середнє значення оригінального списку.
 3)Виведе список, де кожен елемент це значення оригінального списку, яке помножене на середнє значення оригінального списку.
 4)Виведе список, де кожен елемент це значення оригінального списку, з якого було взято квадратний корінь.
